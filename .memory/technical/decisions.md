---

## Rules

- Decisions are **immutable** — don't edit history, add new entries
- Status values: PROPOSED → ACCEPTED → SUPERSEDED → REJECTED
- Rationale must explain **why this over alternatives**

## How to add

1. Add row with date and short description
2. Fill in rationale (the most important field)
3. Set status to PROPOSED initially
4. Update to ACCEPTED once implemented
